Data Types, Functions and Operations


Welcome back. I'm glad you made it to the second Chapter.
In this Chapter we will have a closer look on what we did the last time, but first some theories on Data Types.
Java Script handles Data Types implicit, but that does not mean they don't exist here as well. Some of you might ask now what are even Data Types? Data Types are some what an agreement on which form a given variable or data has to be, to perform operations with them. There are 6 basic Data Types in Java Script:

    • Integer (123456)
    • Double (123.456)
    • Float (123e15)
    • Short (as Integer but max:)
    • Long (as Integer but max:)
    • String ("Hello World")
On these are operations defined. The Basic Operations are:

    • +
    • -
    • /
    • *
    • %
    • .
